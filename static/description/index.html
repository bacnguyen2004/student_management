Student Management

Odoo 18 portfolio module for training centers

Manage students, teachers, courses, classrooms, enrollments, grades, reports, mail notifications, scheduled jobs, and role-based access.

Highlights

  • Student, Teacher, Course, Classroom, and Enrollment models
  • Enrollment workflow: Draft, Studying, Completed, Cancelled
  • List, form, kanban, calendar, graph, and pivot views
  • Bulk enrollment wizard for fast course registration
  • Teacher, Staff, and Administrator security groups
  • PDF reports for student cards, courses, and class grades
  • Email templates for enrollment notifications
  • Daily cron to auto-complete eligible enrollments

Built For Demo

The module includes demo data and focused tests, making it easy to install, explore, and present as a complete Odoo learning project.
